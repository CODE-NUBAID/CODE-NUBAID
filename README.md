## 🚀 Featured Projects

### 🧾 InvoiceAI — Financial Document RAG Assistant

[![View Repo](https://img.shields.io/badge/⭐%20View%20Repo-Financial%20Document%20RAG-58A6FF?style=for-the-badge\&logo=github\&logoColor=white)](https://github.com/CODE-NUBAID/Financial-Document-RAG-Assistant)

A multi-document RAG app for invoice PDFs with source-grounded answers, exact page/file tracing, and a hard refusal when the answer is missing.

<details>
<summary><b>Architecture highlights</b></summary>

* FAISS incremental indexing with multi-PDF support
* Per-user session isolation
* Per-route rate limiting
* Anti-hallucination prompt contract
* Structured evaluation harness for retrieval and answer quality

</details>

<br/>

### 📈 Store Manager AI — Retail Intelligence Dashboard

[![View Repo](https://img.shields.io/badge/⭐%20View%20Repo-Store%20Manager%20AI-58A6FF?style=for-the-badge\&logo=github\&logoColor=white)](https://github.com/CODE-NUBAID/Store-Manager-AI)

A conversational retail forecasting system that replaced a weaker baseline with a feature-engineered XGBoost regressor and a ReAct-style agent interface.

<details>
<summary><b>Architecture highlights</b></summary>

* XGBoost regressor for next-day sales forecasting
* ReAct tool-calling agent
* Leakage-aware feature engineering
* Time-based 85/15 evaluation split
* Flask REST endpoints for charting, metrics, and training

</details>
